# Qwen 3.5 4B

## Model Info
- **Architecture:** Qwen3
- **Parameters:** 4B
- **Quantization:** Q4_K_M
- **Vision:** Yes (mmproj BF16)

## Files Required
- weights/qwen3.5-4b-q4_k_m.gguf
- vision/mmproj-BF16.gguf

## Usage Notes
Place model weight files in the weights/ and vision/ subdirectories.
Run scripts/validate_models.ps1 to verify checksums after download.
